# MUX-Based Combinational Circuits

## Description
This repository contains the Verilog implementation of basic combinational circuits using Multiplexers (MUX). Each circuit includes both the design code and its testbench.

## Circuits
- Half Adder (HA)
- Full Adder (FA)
- Half Subtractor (HS)
- Full Subtractor (FS)

## Tool Used
- Verilog HDL
- EDA Playground
